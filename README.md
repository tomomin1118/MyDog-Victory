# MyDog-Victory

愛犬写真投稿アプリ

- 一定期間の評価数をカウントし、メインページでトップ５を公開。
- 飼い主だからこそ撮れる可愛い一瞬やカッコいい瞬間を捉えた写真を、共有して楽しめる。

- インスタなどのＳＮＳでのペット写真の投稿が多い背景には、愛犬を見てもらいたいという気持ちがある。
- また、飼ってなくても見ることで癒される効果もあり、幸せな空間を作る。
